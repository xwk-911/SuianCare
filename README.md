## 它能做什么

- **主动提醒，无需开口**：第一视角持续观察，模型主动判断"此刻是否需要提醒"，语音 + 震动提示。
- **覆盖日常盲区**：忘关火、忘吃药、出门忘带钥匙/忘关门窗……
- **个性化、越用越懂**：上传你关心的事即可定制任务；一次反馈，模型就修正、更懂你的父母。
- **省电**：场景无关时自动降低观察频率。
- **隐私优先**：原始画面不出本地——本地模型先"风格化"，只上传保留语义、抹去身份的画面。

## 形态

手机竖插胸前口袋、后置摄像头露出朝前 → 平时口袋里默默观察 → 需要时震动+语音 → 掏出手机即见提醒。
